## ğŸ³ Running the Application (MANDATORY)

### Prerequisites

* Docker Desktop (Linux containers enabled)
* Git

### Clone the Repository

```bash
git clone https://github.com/basasindhu04/multi-tenant-saas.git
cd multi-tenant-saas
```

### Start All Services

```bash
docker-compose down -v
docker-compose up -d
```
